
# 1장 통계학의 기초개념

1. 통계학이란 어떤 학문인가?
- 통계학의 정의
    
    : 통계학이란 불확실한 상황에서 현명한 의사결정을 하기 위한 이론과 방법의 체계이며, 통계학은 자료의 수집/분류/분석과 해석의 체계를 갖는다
    
1. 통계학의 기본 용어
- 모집단 : 연구자의 관심의 대상이 되는 모든 개체의 집단
- 표본 : 모집단에서 조사대상으로 채택된 일부
    - 표본을 선택, 분석하여 모집단의 특성을 파악하려는 것이 통계학의 주류
- 모수 : 모집단의 특성을 수치로 나타낸 것
    - $\mu, \sigma^2, \sigma$ = 산술평균, 분산, 표준편차 → 모수
    - $N$ = 모집단의 관찰 수
- 통계량 : 표본의 특성을 수치로 나타낸 것
    - $\bar{X}, S^2, S$ = 산술평균, 분산, 표준편차 → 통계량
    - $n$ = 표본의 관찰 수
- 기술통계학 : 자료수집, 자료정리, 자료해석의 과정을 통해서 표본집단의 특성을 파악
- 추측통계학 : 표본에서 얻은 통계량을 기초로 해서 모집단의 특성을 추론
    1. 모수통계학 : 모집단의 분포모양이 정규분포라는 가정이 필요, 연속적  자료 사용
    2. 비모수통계학 : 모집단의 분포모양 가정이 필요없고, 비연속적 자료를 사용
